# Personal Portfolio Website

A modern and responsive personal portfolio website built with **React** to showcase my skills, projects, and contact information.

## 🚀 Live Demo

Coming Soon...

## 📌 Features

- Responsive design for desktop, tablet, and mobile
- Home / Hero section
- About Me section
- Projects showcase
- Contact form
- Clean component-based architecture
- Fast development with React + Vite

## 🌙 Stretch Features

- Dark mode toggle
- Smooth animations and transitions
- Contact form validation
- Resume download button

## 🛠️ Tech Stack

- React
- JavaScript
- Vite
- Sass (SCSS)
- CSS3
- Git
